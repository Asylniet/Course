Курстың мазмұны

HTML

  1. 1. Visual Studio Code код редакторымен танысасыз
  2. 2. Ол редакторда қалайша код жазуды 2 есе тездетуге болатынын білесіз
  3. 3. Блок элементтерімен, мәтінмен, формалармен, сілтемелермен, сурет, видео, аудиомен, атрибуттармен жұмыс жасайсыз
  4. 4. Сайттың дизайнны құрастыру, оны жақсарту, қателіктерді талдау жайлы арнайы жеке сабақ аласыз

CSS

1. Селектор, элемент моделі, flexbox жүйесін үйренесіз
2. Мәтінді стильдейсіз, блокты әсемдейсіз
3. display, position, transform сипаттамаларына толық түсінік аласыз
4. Жайттармен жұмыс жасайсыз
5. Анимацияны үйренесіз
6. px, %, em, rem, ch, vw, wh өлшемдерімен танысасыз
7. responsive design(сайттың барлық құрылғыда дұрыс көрінуі) техникасын үйренесіз
8. медиа-сұраныс (медиа-запрос) не екенін білесіз
9. иконкалармен жұмыс істейсіз

Javascript

1. Элементтерді таңдауды, оларға функция қолдануды үйренесіз
2. Элементтің классына әсер етуді білесіз
3. Формалармен жұмыс істейсіз
4. Regex тілі арқылы жазылған email, құпиясөзді тексеруді үйренесіз
5. Тексерістің нәтижесіне байланысты ақпаратты өзгерте алатындай боласыз
6. Элементке байланысты жайттарға қатысты функция орындайсыз
